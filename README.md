## Open Source Contributions

### [larksuite/cli](https://github.com/larksuite/cli) — Lark Official CLI Tool

**[PR #740](https://github.com/larksuite/cli/pull/740): Migrate task shortcut errors to structured output for AI agent compatibility**

Migrated 16 error paths in the task management module from bare `fmt.Errorf` to structured `output.ErrValidation`/`output.Errorf`, enabling AI agent consumers (Claude Code, Cursor, Gemini CLI) to programmatically parse and recover from CLI errors via JSON error envelopes. Categorized errors into validation (exit 2) vs API (exit 1) types with actionable hints. Added unit tests with structured error assertions across 4 source files and 3 test files.
